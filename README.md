# 学习搭建cli脚手架

## 安装
### 全局安装
$ npm install -g mpy-cli
# or yarn
$ yarn global add mpy-cli

### 借助npx
创建模版
$ npx create mpy-cli <name> [-t|--template]
示例
$ npx create mpy-cli hello-cli -template dumi2-demo

## 使用
创建模版
$ mpy-cli create <name> [-t|--template]
示例
$ mpy-cli create hello-cli -t dumi2-demo
